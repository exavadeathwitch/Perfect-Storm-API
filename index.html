How to find the root folder for NS: UNS4 on Steam

  1. Open Steam and go to your game library
  2. Right-click Naruto Shippuden: Ultimate Ninja Storm 4 from the game's list
  3. Under the "Manage" list, click "Browse local files"

Installation Instructions

  1. Download the latest release of Perfect Storm
  2. Extract the zip file
  3. Copy the contents of the folder
  4. Paste the copied files into your root Naruto: Storm 4 folder

How does it work?

This mod utilizes a tool called "Modding API for Naruto Shippuden: Ultimate Ninja Storm 4", created by Zealot Tormunds. It uses .DLL Code Injection to hook onto the game's active memory stored in RAM. From there, an Assembly Language layer of code allows us to modify the game's active memory in realtime. By making use of C++ and Lua, we can allow for high-level code to read and control low-level operations.

This is in direct contrast to traditional Storm modding, which utilizes a data_win32 folder to overwrite hex (compiled) code present in the game's filesystem at a low level.
