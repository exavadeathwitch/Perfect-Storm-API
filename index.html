The definitive version of Naruto Shippuden: Ultimate Ninja Storm 4.

As CC2 has neglected to fix bugs and balance characters as they are not focused on providing a competitive experience, we are taking this into our own hands. We want to provide an experience founded in the greatest aspects of Naruto Shippuden Ultimate Ninja Storm 4 while incorporating new gameplay ideas.

Light Version strives to rebalance on a granular level, retaining the core fundamentals of Storm. This is aimed towards current Storm players that are looking for the fundamentals they're accustomed to, or a transitional period to Dark Version.

Dark version strives to implement radical changes to the system that allow for a higher level of play to be present. This is aimed towards both current and new Storm players that are looking for new gameplay or higher levels of competition.

Installation

  1. Download the latest release of Tournament Edition
  2. Copy the .DLL release file in your root Naruto: Storm Rev folder

How to find the root folder for NS: UNSR on Steam

  1. Open Steam and go to your game library
  2. Right-click Naruto Shippuden: Ultimate Ninja Storm Revolution from the game's list
  3. Under the "Manage" list, click "Browse local files"

How does it work?

This mod utilizes a tool called "Modding API for Naruto Shippuden: Ultimate Ninja Storm Revolution", created by Zealot Tormunds. It uses .DLL Code Injection to hook onto the game's active memory stored in RAM. From there, an Assembly Language layer of code allows us to modify the game's active memory in realtime. By making use of C++ and Lua, we can allow for high-level code to read and control low-level operations.

This is in direct contrast to traditional Storm modding, which utilizes a data_win32 folder to overwrite hex (compiled) code present in the game's filesystem at a low level.
